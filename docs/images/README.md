# README screenshots

The main `README.md` references three images from this folder. Capture them from the Server demo
(`dotnet run --project demo/MudKeyboard.Demo.Server`) at a comfortable width and drop them in with
these exact names:

| File | What it should show |
| --- | --- |
| `keyboard.png` | The **Full QWERTY** section — the inline `<MudKeyboard>` with the bound text field above it. |
| `numpad.png` | The **Numpad** and/or **Pricepad** sections (calculator pad + `£` currency entry). |
| `docked.png` | The **global docked keyboard** slid up from the bottom after focusing a field. |

Tip: capture both light and dark mode (toggle in the app bar) if you want to show off the automatic
theming — a side-by-side makes a nice hero image.
